Machine Learning Process

1. Data Pre-processing
The data sources were as follows with the files downloaded from each in text format.

  • k-mer counting: Long DNA sequences are broken into k-mer length overlapping “words” and then created Words model using CountVectorizer(). Example: 6 (hexamers), “ATGCATGCA” becomes: ‘ATGCAT’, ‘TGCATG’, ‘GCATGC’, ‘CATGCA’.
  • In genomics, to these types of manipulations are called as "k-mer counting", counting the occurrences of each possible k-mer sequence and can be realised by using Python NLP natural language processing tools.
2. Data Modeling using Machine Learning libraries
  1. FIT: Process DNA sequence datasets for input features/ vectors of uniform length
  2. TRAIN: Build classification model Trained on the human DNA sequence TO predict a gene family
  3. PREDICT: Test model with human and chimpanzee DNA sequence Datasets
  4. PRECISION & RECALL: compare the accuracies by Creating Confusion matrix to compare predicted Vs actual values
    5. 3. Precision and Recall
    Tested on human DNA and Chimpanzee DNA sequence datasets and compare accuracies
  5. By creating Confusion matrix to describe performance of model 
  6. Using sklearn.metrics to get Accuracy, Precision, Recall, F1)
